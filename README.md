# Advanced Topics in Machine Learning and Optimization Project

</div>

Project for the "Advanced Topics in Machine Learning and Optimization Project" course at University of Trento.

## Installation and Execution
Follow the following steps to run the project code:

1. Clone the repository:
   ```sh
   git clone https://github.com/Frasor2002/ATMLO_Project.git
   cd ATMLO_Project
   ```

2. Install modules:
   ```sh
   pip install -r requirements.txt
   ```

3. Run the XIL loop:
   ```sh
    python main.py
   ```
   The jupiter notebook `project.ipynb` contains the code to replicate all the steps in the report.
